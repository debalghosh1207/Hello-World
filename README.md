# Hello-World

A simple C++ project demonstrating CMake build system integration with Yocto/Bitbake.

## Features
- `say_hello()`: Prints "Hello"
- `say_hi()`: Prints "Hi"
- `say_hello_world()`: Prints "Hello World"

## Building
```bash
mkdir build && cd build
cmake ..
make
./helloworld
```
